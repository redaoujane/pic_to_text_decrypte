Arabic Document Digitizer (OCR) 
Developed by Reda Oujane (trojan)

A robust Optical Character Recognition (OCR) tool specifically optimized for the Arabic language. Unlike standard OCR APIs, this project utilizes PaddleOCR to process images locally, eliminating API quotas, limits, and privacy concerns.

 Key Features:
100% Offline & Secure: No data is sent to third-party APIs (like chatgpt - gemini ...).

Smart RTL Correction: Custom Python algorithms to fix Arabic right-to-left mirroring and word ordering issues common in standard OCR engines.

Intelligent Bounding Boxes: Sorts and groups text line-by-line horizontally, preserving the document's original structure.

Modern UI: Built with Streamlit for a fast, responsive, and user-friendly experience.
